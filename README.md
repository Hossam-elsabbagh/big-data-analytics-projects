# Big Data Analytics Projects

This repository contains three Big Data Analytics mini projects.

## Projects

### Mini Project 1 - The Digital Librarian
Distributed reverse indexing using Hadoop, HDFS, and MapReduce.

### Mini Project 2 - Spark Case Study
Large-scale analytics using Spark RDDs, DataFrames, and Spark SQL.

### Mini Project 3 - Real-Time Recommendation System
Batch ALS recommendation model integrated with Kafka and Spark Structured Streaming.
